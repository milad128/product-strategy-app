From old lifecycle.html (browser storage)

Positions saved in file:///…/bnpl-lifecycle/lifecycle.html live in that browser’s localStorage. The server cannot read them directly.

  1. Ensure this app is running on http://127.0.0.1:8000
  2. Open Desktop/vibe coding/bnpl-lifecycle/import-to-database.html in the same browser you used for lifecycle.html
  3. Click Import to database

Only works if you previously saved lifecycle data while using this app on localhost.

Paste JSON manually

Paste the contents of bnpl-lifecycle-layout from browser DevTools → Application → Local Storage.
