# ARIA Game — PROTOCOL: OVERRIDE Squads discover secret mission URLs and submit answers via HTTP. ## Digit submission (missions 1-5) ```bash curl -X POST $HOST/api/mission/digit \ -H 'Content-Type: application/json' \ -d '{"squad":"VIPER","digit":"8"}' ``` ## Team registration (step 2) ```bash curl -X POST $HOST/api/act0/register \ -d '{"teams":[{"name":"VIPER","members":["Ada","Bo"]}]}' ``` ## Passphrase override Use the mission slug from room clues. Body uses `team` and `keyword` fields (not documented here — see source). ```json { "team": "VIPER", "keyword": "HUMANS" } ``` ## Game Master - `/gm/timer/start` — start mission clock - `/gm/reset-act` — clear mission progress
